# prismagram

Instagram clone with Express + Prisma + React and React Native

## User stories

- [x] Create account
- [x] Request secret
- [x] Confirm secret (Log in)
- [x] Like / Unlike a photo
- [x] Comment on a photo
- [x] Search by user
- [x] Search by location
- [x] Follow User
- [x] Unfollow User
- [x] See user profile
- [x] See my profile
- [x] Edit my profile
- [x] See the full Photo
- [ ] Upload a photo
- [ ] Edit the photo (Delete)
- [ ] See the feed
