# prismagram

Instagram clone with Express + Prisma + React and React Native

## User stories

- [x] Create account
- [x] Request secret
- [x] Confirm secret (Log in)
- [x] Like / Unlike a photo
- [x] Comment on a photo
- [x] Search by user
- [x] Search by location
- [x] Follow User
- [x] Unfollow User
- [x] See user profile
- [x] See my profile
- [x] Edit my profile
- [x] See the full Photo
- [x] Upload a photo
- [x] Edit the photo (Delete)
- [x] See the feed
- [x] Send private Message
- [x] See rooms
- [x] See room
- [] Receive Message
